TWEEZ

Site de jogos online desenvolvido como trabalho de conclusão de curso

Introdução

Nosso projeto, TWEEZ, surgiu com o objetivo de oferecer uma plataforma simples, intuitiva e visualmente agradável para que os usuários possam acessar diferentes jogos em um só lugar.

Ao longo do projeto, buscou-se criar um site funcional, organizado e com recursos como páginas individuais de jogos, sistema de favoritos e interface responsiva.

Objetivos ao longo do projeto

    Ao longo do projeto, metas precisaram ser estabelecidas para tornar seu processo de desenvolvimento eficiente e prático.

  • Desenvolver o layout em HTML e a estrutura visual do site em CSS.
  • Criar páginas individuais para cada jogo com descrição e iframe. (Processo que logo em seguida foi automatizado)
  • Implementar sistema de favoritos, login e senha com JavaScripte PHP
  • Organizar os jogos por categorias para fácil acesso.
  • Garantir padronização visual entre todas as páginas, as tornando acessíveis e sem distorções em qualquer dispositivo

POR QUE um site de jogos?

    Por que, entre tantas possibilidades de projetos, nossa equipe escolheu desenvolver um site de jogos, apesar de já existirem alguns disponíveis pela internet?
    Jogos digitais permanecem populares até os dias de hoje, devido a sua facilidade para serem jogados, diretamente do navegador web do usuário.

    Portanto, nós percebemos que os sites brasileiros disponíveis atualmente possuem designs bagunçados, muitos anúncios e pop-ups desnecessários, não são muito intuitivos, e alguns jogos nem ao menos funcionam. Dito isso, viu se necessária a necessidade de desenvolver uma plataforma que contornasse todos esses obstáculos.

Metodologia

  • Pesquisa inicial sobre sites de jogos, como funcionam, como usar legalmente e referências visuais
  • Criação do protótipo, definindo cores, layout, organização e identidade visual
  • Estruturação do site em HTML, iniciando pela interface principal.
  • Estilização em CSS, seguindo o design anteriormente definido no protótipo
  • Implementação de funcionalidades JavaScript e PHP, como o perfil de usuário, sistema de filtros, login
  • Testes práticos, ajustes de embed, correções, feedbacks de usuários testando.

Ferramentas

  • HTML5: estrutura das páginas.
  • CSS3: estilização e identidade visual.
  • JavaScript + PHP: funcionalidades interativas e sistema de favoritos.
  • Figma: criação do protótipo e organização visual.
  • Infinity: Hospedagem do site e banco de dados
  • My SQL + PhpMyAdmin + Node.js

    • Identidade Visual

    Para representar a nossa marca, optamos por utilizar como principais as cores preto e branco. Escolhemos uma fonte retro que remete a jogos para o site, sempre mantendo o minimalismo e simplicidade que o projeto tem o objetivo de transmitir. Nossa logo, ja apresentada previamente, foi desenvolvida por nós mesmas, para assim que o usuário 'bater o olho', se lembrar de nosso site.

    Dificuldades ao longo do projeto

    Ao longo do projeto, apesar da nossa organização ter nos ajudado a evitar obstáculos, enfrentamos inevitaveis desafios, e entre eles podemos citar: Criação de cada tela dos jogos individualmente, o que nos levou a iniciar a pesquisa e aprender como automatizar isso utilizando JavaScript e o Node.js, alguns jogos por iframe deixando de funcionar, responsividade em diferentes tamanhos de tela e tempo limitado para testes e ajustes do site.

    Conclusão

    Apesar dos obstáculos que surgiram ao longo do processo, o TWEEZ representou uma experiência completa de desenvolvimento web, envolvendo planejamento, prototipagem, programação e documentação. O trabalho demonstrou como diferentes áreas da tecnologia podem ser integradas para construir um produto funcional e visualmente consistente.

    O resultado final é um site moderno, interativo e preparado para receber novas funcionalidades e jogos futuramente.
